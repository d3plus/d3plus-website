Getting Your Environment Setup

Step 1.

Download D3plus v0.9

Step 2.

Run a Local Server

Because you will be testing these files locally, your browser will raise errors when trying to complete AJAX requests (which we will use to load our data files). The best way around this is to run a local server from the example folder you have just downloaded. If you have Python installed, use the command line to change into the directory of the project folder you just downloaded and run this command:

python -m SimpleHTTPServer 8888 &

or for Python 3+:

python -m http.server 8888 &

Another alternative is using MAMP (on OSX) or WampServer (on Windows), which will install a local version of the Apache web server.

Step 3.

Test in Browser

Once your server is running, you should be able to go to http://localhost:8888/ and see your first Tree Map!
