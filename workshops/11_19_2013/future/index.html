Down The Road

D3plus was built for a specific project - dataviva.info and as such does not contain every possible visualization imagined. We are hoping to continue expanding D3plus' set of capabilities, but to truly make it a useful tool we will need help from the rest of the community. We hope that many of you in this workshop will use D3plus for your next web based visualization project and submit bugs to us!

On the ToDo List:

  • Make it faster! Optimize.
  • Create a universal circle drawing class (used in many visualizations)
  • Create a more generalized "chart" visualization that would house the stacked, pie scatter and lead to the possibility of a bar and line chart.
  • Improved calling structure, abstract away the requirement of needing to re-call the visualization on the container element every time an attribute is changed.
  • Create a "group" class to allow all visualizations created within a group to listen for the same events and update together
  • Continue work on documentation
  • Create more examples

Where You Come In...

As previously mentioned we hope that all of you go home after this workshop and fork D3plus with your own improvements and submit us tons and tons of pull requests. If you feel intimidated or think D3plus is fine as it is, you could also help us with documentation. We are doing our best to update the wiki but there are certainly holes. We've also created a D3plus google group as a direct line to the developers and other community members.
