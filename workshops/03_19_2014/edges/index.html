Attribute Formatting

Simple Attributes (Array)

When you have multiple data points that may all have similar attributes, it becomes more efficient to pass attribute data that many data points can reference. In this example, the attribute data is in a similar format to the raw data, and they are matched at a specific key.

Simple Attributes (Dictionary)

Alternatively, attribute data can also be passed as a dictionary, keyed by the matching ID.

Complex Attributes

If you are managing multiple IDs (ex. nesting), it is also possible to pass multiple attribute lists, keyed by the appropriate IDs. This example shows Attribute Arrays, but can also be used with Attribute Dictionaries.
